# Sushi plate calculator

## Introduction

This is an app that will help you easily calculate the cost of your sushi meals at a conveyor belt sushi restaurant. Simply input the number of plates you've eaten, and the app will compute the total cost based on the price per plate. Perfect for keeping track of your dining expenses!

If you want to contribute, please refer to the [contributing guidelines](./CONTRIBUTING.md) of this project.

## Live Demo

Visit https://sushi.chatree.dev to see the live demo!
